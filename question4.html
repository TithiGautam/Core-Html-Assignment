HTML TAGS:

  1. h1 to h6: Heading tags
  2. p: paragraph tag
  3. a: anchor tag ,create hyperlink to other web pages
  4. br: inserts a line break within the text
  5. ul: creates an unordered list
  6. img: use to embed image
  7. sub: renders text as subscript
  8. small: reduces the text size
  9. form: used to create af orm
  10. link: used to link external resources to a web page
  11. table: this is used for creating table
